ビンゴゲーム プロジェクト詳細 (開発中)

プロジェクト概要

Node.js, Express, Socket.IO, TypeScript, PostgreSQL を使用して作成中のリアルタイムビンゴゲームです。このプロジェクトでは、特にリアルタイム双方向通信の実現に注力しました。

主な機能 (現時点)

  • **リアルタイム抽選:** サーバーで抽選されたビンゴ数字を、同一部屋の全クライアントにリアルタイムで同期表示。
  • **ルームシステム:** 各ゲームが独立して進行するよう、異なる部屋への参加・管理機能。
  • **ビンゴカード生成:** クライアント側でのランダムなビンゴカード生成と表示。
  • **数字のマーク:** 抽選された数字に対応するカード上のセルを視覚的にマーク。

使用技術

  • バックエンド: Node.js, Express.js, Socket.IO, TypeScript, Prisma ORM, PostgreSQL
  • フロントエンド: HTML5, CSS3, JavaScript (ES6+), Fetch API
  • デプロイ: Render.com, GitHub

工夫した点・学んだこと

  • **Socket.IO:** リアルタイム性の要求されるアプリケーションにおいて、Socket.IO を用いた双方向通信の設計と実装を経験しました。特に、ルーム機能を使って複数のゲームセッションを独立して管理するロジックを構築しました。
  • **TypeScript:** 大規模なプロジェクトにおける型安全性の重要性と、開発効率向上への寄与を実践的に学びました。
  • **データベース設計:** 抽選済み数字の履歴管理など、ゲームの状態をデータベースで永続化する設計を行いました。
  • **デバッグと問題解決:** Socket.IO の通信エラーや DOM 操作のエラーなど、様々な問題をコンソールログや開発者ツールを駆使して解決しました。

今後の展望 (アピールポイント)

  • ビンゴ判定ロジックの実装と、ビンゴ成立時のゲーム終了処理。
  • チャット機能など、プレイヤー間のインタラクションの追加。
  • ユーザー認証機能の導入と、個人成績の保存。
